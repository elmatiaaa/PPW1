Install dan Import Library#

Library yang digunakan yaitu :

  1. Requests –> library untuk mengirim permintaan HTTP di Python dengan cara yang sederhana dan mudah dipahami.

  2. Beautifulsoup4 –> library Python yang memudahkan ekstraksi informasi dari dokumen HTML dan XML. Berguna untuk melakukan web scraping dan parsing data dari halaman web.

  3. Pandas –> library untuk analisis data. Ini digunakan untuk membaca, mengolah, dan menganalisis data dalam format yang disebut DataFrame.

  4. CSV –> library untuk membaca dan menulis data dari atau ke file CSV.

Crawling Berita#

Crawling berita kali ini terdapat 3 kategori:

  1. Otomotif

  2. Megapolitan

  3. Tren yang terjadi

Cara crawling yang saya lakukan disini yaitu dengan mengambil judul dan isi beritanya saja. Untuk mengambil judul dan isi berita disini dapat diproses melalui class yang ada pada elemen web berita online.

Setiap kategori memiliki halaman berita yang berbeda-beda, dapat dimasukkan pada variabel n yang sudah saya sediakan.

Otomotif#

header={
    'user-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/110.0.0.0 Safari/537.36'
}

linkberita = 'https://otomotif.kompas.com/'
listpolitik = []
n = 41

for halaman in range(1, n+1):
  req = requests.get(linkberita)
  soup = BeautifulSoup(req.text, 'html.parser')
  items = soup.findAll('div', 'article__list clearfix') #class semua berita
  for it in items:
    link = it.find('a','article__link')['href'] #class perberita

    req2 = requests.get(link)
    soup2 = BeautifulSoup(req2.text, 'html.parser')
    judul = soup2.find('h1', 'read__title').text    #class judul berita
    isi = "".join(soup2.find('div', 'read__content').text.split('\n'))

    listpolitik.append([judul,isi])

header={
    'user-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/110.0.0.0 Safari/537.36'
}

urlberita = 'https://www.kompas.com/tren?source=navbar'
beritatren = []
n = 50

for halaman in range(1, n+1):
  req1 = requests.get(urlberita)
  soup1 = BeautifulSoup(req1.text, 'html.parser')
  items = soup1.findAll('div', 'trenLatest__item clearfix') #class berita
  for it in items:
    link = it.find('a','tren__link')['href'] #class perberita (pada bagian judul)

    req2 = requests.get(link)
    soup2 = BeautifulSoup(req2.text, 'html.parser')
    judulF = soup2.find('h1', 'read__title').text    #class judul berita
    isiF = "".join(soup2.find('div', 'read__content').text.split('\n'))

    beritatren.append([judulF,isiF])

tren = pd.DataFrame(beritatren, columns=['Judul', 'Isi'])
tren['Label'] = 'tren'
